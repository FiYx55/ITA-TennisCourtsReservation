# Web Client

Web client for the Tennis Court Reservation System.

## Technology

- **Language**: TypeScript
- **Framework**: Next.js / React
- **Interface**: User-facing UI

## Description

The web client provides the user interface for browsing courts, managing profiles, and making reservations. It communicates exclusively through the API Gateway.

## Ports

| Port | Description     |
|------|-----------------|
| 3000 | HTTP (Next.js)  |

## Running with Docker

```bash
docker compose up web-client
```
